# **GraphicVisualization**

中国地质大学（武汉），面向对象程序设计/数据结构与算法课程设计代码，使用 C++ 语言编写的桌面应用。

本项目包含了一系列的代码文件，用于实现图形可视化功能。

## 项目结构

- **代码文件**：所有的代码文件都存放在 `GraphicVisualization` 目录下。
- **代码流程图文件**：代码文件流程图文件位于 `Render.ipynb`。
- **Doxygen 文档**：使用 Doxygen 生成了代码的文档，你可以在 `Doc/html/index.html` 找到它。
- **Doxygen 配置文件**：生成 Doxygen 文档的配置文件位于 `Doc/Doxyfile`。

请确保你已经安装了 Doxygen 与 Graphviz，这样你就可以在本地生成和查看代码文档了。

## 如何使用

1. 打开 `GraphicVisualization` 目录，你会看到所有的代码文件。
2. 如果你想查看代码流程图，打开 `Render.ipynb` 文件即可。
3. 如果你想查看代码文档，打开 `Doc/html/index.html` 文件即可。
4. 如果你想自己生成代码文档，你可以修改 `Doc/Doxyfile` 配置文件，然后运行 Doxygen。
